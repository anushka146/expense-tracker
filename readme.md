# Expense Tracker Web App

A simple web-based Expense Tracker built with **HTML, CSS, and JavaScript**.  
This app helps users track their income and expenses in real-time, calculate balance, and persist data using browser **LocalStorage**.

---

## Features

- Add **Income** and **Expense** transactions
- Assign **category** to expenses
- View **current balance**, total income, and total expenses
- **Delete** any transaction
- **Persistent data** using LocalStorage
- Simple, interactive UI using DOM manipulation

---

## Technologies Used

- **HTML** – for structure and form inputs
- **CSS** – for styling the list and buttons
- **JavaScript** – for functionality, calculations, and LocalStorage

---

## How It Works

1. **User Input:**  
   - Enter amount  
   - Select type: Income / Expense  
   - Choose category (for expense)

2. **Add Transaction:**  
   - Click **Add** button  
   - Transaction appears in the list  
   - Totals update automatically

3. **Delete Transaction:**  
   - Click **X** button next to a transaction  
   - Transaction is removed  
   - Totals update

4. **Persistence:**  
   - All transactions are stored in **LocalStorage**  
   - Data remains even after page refresh
